Zarf — Enterprise B2B SaaS Expense Platform

Zarf (derived from the Arabic word for envelope or vessel) is a premium, enterprise-grade B2B SaaS corporate expense management platform. It is engineered specifically for Small and Medium Enterprises (SMEs) operating in the GCC and United Arab Emirates (UAE), addressing localized compliance, VAT laws, and multi-currency mechanics.

Key Features & Innovations
  • GCC Compliance & VAT Mechanics: Strict UAE Tax Registration Number (TRN) validation and 5% VAT calculations with automatic tax receipt filing.
  • Stateful Shell Tab Caching: Integrated StatefulShellRoute.indexedStack in Flutter routing to cache active screens, keeping transitions lag-free (0ms) and preventing duplicate API network requests.
  • Dynamic Adaptive UI: Checks role permissions on login to morph the mobile navigation from 3 tabs (Employees) to 4 tabs (Managers/Admins), adding a dynamic approvals view with a live pending review badge counter.
  • AI Cost-Sentry & Rate Limiting: Hardened Node.js API with custom aiRateLimiter (10 requests per 15 minutes per IP), 5MB file upload limits, and production error sanitization to protect paid LLM keys.
  • Advanced Security Hardening: Configured Express middleware (HSTS SSL enforcement, SAMEORIGIN clickjacking block, nosniff MIME protection, and strict CSP headers).
  • CI/CD & DevOps Automation: Bash pipeline (build_apk.sh) to automatically rename, build, and version release APKs based on `pubspec.yaml` semver regex extraction. Custom Vercel routing rules matching client-side React routes to eliminate 404 errors.

Raha — Expat Food & Services App

Raha is a consumer mobile app for expats and immigrants living in the Gulf. It helps users discover food from their home cuisine, book trusted home services, and get AI-personalised recommendations tailored to their nationality, city, and interests — across Dubai, Jeddah, Riyadh, and Abu Dhabi.

Key Features & Innovations
  • Nationality-Aware Onboarding: Captures nationality, city, neighbourhood, and interests, feeding directly into the AI recommendation engine.
  • AI Recommendations: Gemini 2.0 Flash generates personalised home recommendations server-side — nothing hardcoded in the app.
  • Live Food Data: A backend sync script pulls real restaurant data from the Google Places API across 9 cuisines (Indian, Kerala, Punjabi, Filipino, Pakistani, Lebanese, Saudi, Gulf, Emirati), replacing static seed data.
  • Photo Proxy Security: Google Places photos are streamed through a backend proxy endpoint, so the Places server key never reaches the client — even a decompiled APK reveals nothing.
  • Home Services Booking: Provider discovery, detail view, booking flow, and booking history with cancellation.
  • Cold-Start Mitigation: A GitHub Actions workflow pings the Render backend every 10 minutes to reduce cold-start latency on the free tier.
